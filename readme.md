# Jasstafel PWA

## Dateien
- `index.html` – Die App
- `manifest.json` – PWA Manifest (App-Icon, Name, Farben)
- `sw.js` – Service Worker (Offline-Support)

## Hosting (kostenlos)

### Option 1: GitHub Pages (empfohlen)
1. GitHub-Account erstellen auf github.com
2. Neues Repository erstellen (z.B. "jasstafel")
3. Alle 3 Dateien hochladen
4. Settings → Pages → Source: "main" Branch → Save
5. App ist erreichbar unter: `https://DEIN-NAME.github.io/jasstafel/`

### Option 2: Netlify
1. netlify.com → "Add new site" → "Deploy manually"
2. Ordner mit den 3 Dateien reinziehen
3. Sofort online!

## Installation auf Handy

### iPhone (Safari)
1. URL öffnen
2. Teilen-Button (□ mit Pfeil) tippen
3. "Zum Home-Bildschirm" wählen
4. "Hinzufügen" bestätigen

### Android (Chrome)
1. URL öffnen
2. Banner "Installieren" antippen, ODER
3. Menü (⋮) → "App installieren"

## Funktionen
- 4 Spielmodi: Schieber, Differenzler, Coiffeur, Molotov
- Teamnamen anpassbar
- Punktziel einstellbar (1000 / 1500 / 2000 / 2500 / kein Limit)
- Match-Bonus (+100 Punkte)
- Schnelleingabe-Buttons
- Automatische Gegenpunkte (Total 157)
- Rundenhistorie
- Fortschrittsbalken
- Gewinnermeldung
- Letzte Runde rückgängig machen
- Spielstand wird automatisch gespeichert
- Funktioniert offline
